Platform Technologies & Vision

Our Moto: To empower civil engineers with ultra-modern, ambient, intelligent tools that drastically accelerate geographical analysis and hydraulic systems design.

Why is it useful? Instead of using multiple disjointed tools for terrain mapping, calculation, 3D checking, and AI consulting, DrainFlow Pro merges them all into one sleek, browser-based, interactive matrix that feels native, premium, and instantly responsive.

HTML / CSS / JS

The core of the application relies on deeply optimized vanilla JavaScript and custom HTML5/CSS3. It uses a "Glassmorphism" UI design system with dynamic, ambient background rendering and pure CSS Grid geometry.

Groq AI

Powered by llama-3.3-70b-versatile via the Groq API. Provides fast, free AI-powered hydraulic analysis, soil refinement, and chatbot consultation — no credit card required.

Supabase Auth

Integrated directly with a live Supabase PostgreSQL backend. Provides instantaneous authentication logic via JWT (JSON Web Tokens) allowing secure state storage.

Leaflet.js

Powers the interactive Map Designer, providing real-world topographical context and geolocation tagging for specific drainage deployments.

Three.js

A WebGL environment generating dynamic, procedural 3D terrain representation and rendering real-time pipe models and water flow visualizations natively inside the browser.

Chart.js

Used for the Results dashboard to plot complex, high-performance mathematical arrays for velocity and Froude/Reynolds relationships instantly.
